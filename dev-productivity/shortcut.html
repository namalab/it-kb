開発者が知っておくべきショートカット・Tips集

はじめに

IT・特にシステム開発に携わる開発者向けに、知っておくべきショートカットをまとめておきました。
細かいものまで網羅して覚えきれないような物量にするのではなく、「最低限これだけは知っておくと生産性向上に寄与できます」というレベル感で書いています。
生産性向上の一助になれば幸いです。

対象者

  • システム開発に携わる開発者

得られる知識・ゴール

  • 最低限知っておくべき便利なショートカットを覚えることができる
  • 結果として作業の生産性アップが期待できる

汎用 Ctrl+X 切り取り
汎用 Ctrl+V 貼りつけ
汎用 Ctrl+A 全選択
汎用 Ctrl+Z アンドゥ
汎用 Shift+Ctrl+Z または Ctrl+Y リドゥ
汎用 Ctrl+B Bold（太字）
汎用 Ctrl+I Italic（斜字）
汎用 Ctrl+U Underline（下線）
汎用 Ctrl+N 新規作成
汎用 Ctrl+S 保存
汎用 Ctrl+P 印刷
汎用 Ctrl+F 検索
汎用 Ctrl+R 置換
汎用 F11 全画面表示
Windows Win+E エクスプローラーを表示
Windows PrintScreen 画面キャプチャ（画面全体をコピー）
Windows Alt+PrintScreen 画面キャプチャ（ウィンドウをコピー）
Windows Win+L 画面をロック
Windows Win+D デスクトップを表示
Windows Win+M 全ウィンドウを最小化

ショートカット（Windows プラスアルファ）

知っておくとかなり便利！ プラスアルファのショートカット集。

分類 ショートカット 概要・備考
Windows Win+V クリップボードから履歴貼りつけ
Windows Shift+Win+S 画面キャプチャ（画面の一部をコピー）
Windows Win+R 「ファイル名を指定して実行」を開く
Windows Win+S 「ここに入力して検索」を開く
Windows Win+X スタートボタンの右クリックメニューを開く
Windows Win+ALT+R 画面の動画キャプチャ（画面全体を動画として録画）
録画したものはWin＋Gで表示されるXboxゲームバー内で確認できる
Windows Win+W Whiteboard等ペンツールを開く

小技 Tips

以下は Windows での Tips と言える小技です。参考になれば幸いです。

  • Explorer のアドレスバーに「cmd」と入力してエンターキー
    →該当フォルでコマンドプロンプトを表示
  • Explorer でフォルダを選択してShift＋右クリック→「PowerShellウィンドウをここで開く」
    →該当フォルダでPowerShellを表示
  • Explorer でフォルダを選択してShift＋右クリック→「Linuxシェルをここに開く」
    →該当フォルダでWSLシェルを表示
  • Explorer でフォルダやファイルを選択してShift＋右クリック→「パスのコピー」
    →フルパスをコピー（複数ファイルを一括でコピーも可）
  • 「ファイル名を指定して実行」で「.」のみ入力してエンターキー
    →ユーザーディレクトリを開く。「.ssh」と入力してエンターならユーザーディレクトリ\.sshを開く
  • 「ファイル名を指定して実行」で「%appdata%」
    →AppData¥Roamingディレクトリを開く
  • 「ファイル名を指定して実行」で「%AppData%\Microsoft\Windows\SendTo」
    →SendToディレクトリを開く（ここにショートカットを作成することで、右クリックメニュー→「送る」に登録される）
  • コマンドプロンプトやバッチファイルにて wsl コマンドも利用可能
    • 例：wsl grep any *.txt → フォルダ内のテキストファイルから GREP
    • 例：dir /s | wsl grep DIR のように、DOS コマンドと組み合わせることも可能
